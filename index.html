Portfolio

Swarm Key

Swarm Key is an interactive generative music system that utilises swarm intelligence. Western tonal theory and user preference is embedded into the system as both guidance and constraints for swarms. Built in openFrameworks.

Source code available here. Album of composed music can be found here.

Flow Field Series

As a side hobby from university study, I have experimented with different variations of noise and flow fields.

This is an ongoing process and I use a dedicated Instagram and Twitter account, as linked above, to post the results. Experiments have been created using Processing.

Roses of Grandi Series

Another series of work includes exploring the Roses of Grandi algorithm.

This work has also been posted to the dedicated Instagram and Twitter account, and also has been created using Processing.

Step Sequencer

Step sequencer built using openFrameworks. The system allows for different octaves and frequency analysis of the overall sound.

Source code available here.

Frequency Modulation with Bitalino

Frequency modulation using the EMG sensor provided by a BITalino board kit. Using this signal and down-sampled frames of the camera, different oscillator frequencies are controlled using linear regression.

Video of system can be found here.

Euclidean Sequencer

Using Bjorklund's algorithm, 'The Theory of Rep-Rate Pattern Generation in the SNS Timing System', to determine Euclidean rhythms. Built in openFrameworks.

Source code available here.
